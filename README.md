# VOLTAGE SENSOR
Example code for VOLTAGE SENSOR

WARNING maksimum voltage sense is 25V
![alt tag](http://www.emartee.com/Images/websites/emartee.com/Voltage(2).jpg)

## Wiring ##
| Fingerprint   | PIN Arduino UNO |
| ------------- | --------------- |
| +(PLUS)       | 5V              |
| -(MINUS)      | GND             |
| S             | A0/Analog 0     |
